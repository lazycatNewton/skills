# 缠论核心概念定义参考

> 本文档为缠论技能（chan）的 reference 文件，提供顶分型、底分型、背离、背驰、笔、中枢、三类买卖点的严格定义。
> 所有定义均源自缠中说禅《教你炒股票108课》体系，经 /Users/didiapp/dev/chan/缠论笔记_定理框架与核心知识.md 交叉验证。

## 1. 顶分型（Top Fractal）

**定义**：连续三根经过包含处理后的 K 线，中间 K 线的高点为三根中最高，且中间 K 线的低点也为三根中最高。

**结构判别**：

```
设三根K线为 K1, K2, K3（从左至右），则：

K2.high = max(K1.high, K2.high, K3.high)
AND
K2.low  = max(K1.low,  K2.low,  K3.low)
```

**含义**：卖方分力暂时取得胜利。三根K线博弈的结果是多头三次冲击但高点逐次被压制——最终中间K线成为局部最高点后回落。

**铁律**：所有顶部必然是顶分型。没有顶分型就没有顶。

---

## 2. 底分型（Bottom Fractal）

**定义**：连续三根经过包含处理后的 K 线，中间 K 线的低点为三根中最低，且中间 K 线的高点也为三根中最低。

**结构判别**：

```
设三根K线为 K1, K2, K3（从左至右），则：

K2.low  = min(K1.low,  K2.low,  K3.low)
AND
K2.high = min(K1.high, K2.high, K3.high)
```

**含义**：买方分力暂时取得胜利。空头三次打压但低点逐次被抬高——中间K线成为局部最低点后反弹。

**铁律**：所有底部必然是底分型。没有底分型就没有底。

---

## 3. 背离（Divergence）

**定义**：价格走势与技术指标（常用 MACD）之间出现方向不一致的广义现象。价格创新高/新低，但指标未能同步确认。

**与背驰的关系**：

| 概念 | 范畴 | 含义 |
|------|------|------|
| **背离** | 广义技术分析概念 | 价格与指标的任意方向不一致 |
| **背驰** | 缠论专有概念 | 趋势中同向段之间的**力度**衰竭，须满足严格的几何结构条件 |

> 「背驰不是价格的背离，而是**力度的背离**。」背离是信号，背驰是结论。所有背驰都是背离，但不是所有背离都是背驰。

**MACD 计算口径**：

MACD 以连续 K 线的收盘价 `close` 为唯一价格输入，默认采用常见的 12/26/9 参数：

```
EMA12 = close 的 12 周期指数移动平均
EMA26 = close 的 26 周期指数移动平均

DIF = EMA12 - EMA26
DEA = DIF 的 9 周期指数移动平均
MACD柱 = 2 * (DIF - DEA)
```

在图形化或背离辅助观察中，DIF/DEA 可作为黄白线，MACD柱可作为红绿柱。用于背驰力度比较时，应比较对应走势段内 MACD柱的积分面积；单纯价格创新高/新低而 MACD 未确认，只能称为背离信号，不能直接判定为背驰结论。

## 4. 背驰（Exhaustion / Bèi Chí）

**定义**：在一个趋势中（至少包含两个同级别中枢），连接中枢的同向次级别走势段之间出现力度衰竭。标志驱动趋势的内部能量已经枯竭，趋势即将转折。

**标准模型**（趋势背驰）：

```
走势结构：a + A + b + B + c

其中 A、B 为同级别走势中枢，
a、b、c 为连接中枢的次级别走势。
```

**三个必要条件**：

1. **结构条件**：必须存在至少两个同级别中枢 A 与 B（「没有趋势，没有背驰」）
2. **力度条件**：c 段力度 < b 段力度（量化：c 段对应 MACD 柱子积分面积 < b 段柱子积分面积）
3. **MACD 条件**：MACD 黄白线（DIF/DEA）必须回拉至零轴附近，体现中枢 B 的引力场作用

**趋势背驰 vs 盘整背驰**：

| | 趋势背驰 | 盘整背驰 |
|---|---|---|
| **结构** | a+A+b+B+c（双中枢） | a+A+b（单一中枢） |
| **转折级别** | 大概率以上级别反趋势（绝对逆转） | 大概率拉回原中枢（震荡回归） |
| **用途** | 锁定第一类买卖点 | 中枢内高抛低吸（短差） |
| **操作** | 全仓反向操作 | 降成本，不改变整体仓位方向 |

**动态乘2法则**：当柱子伸长的力度变慢时，将已出现的面积乘2，即可在前一根柱子完成前当下判定背驰，无需等待整段走完。

## 5. 笔（Stroke / Bǐ）

**定义**：一个顶分型与一个底分型之间可构成「一笔」。

**结构要求**：

```
笔 = 顶分型 + 底分型
     或
     底分型 + 顶分型
```

**铁律**：一笔必须至少包含 3 个基本 K 线单位。

**划分三步法**（缠师第77课）：

1. 确定所有标准分型
2. 对同向连续分型进行筛选——顶保留最高的，底保留最低的
3. 连接相邻的顶和底，形成笔

> 笔是缠论形态学的最小趋势单元。向上笔 = 从底分型到顶分型；向下笔 = 从顶分型到底分型。

## 6. 中枢（Price Center / Zhōngshū）

**定义**：某级别走势类型中，被至少连续三个次级别走势类型所重叠的价格区间。中枢是多空双方力量达到暂时平衡的区域。

**边界公式**：

```
[ZD, ZG] = [max(低点), min(高点)]

其中（对构成中枢的各段）：
  ZG = min(各段高点)    ← 中枢上沿
  ZD = max(各段低点)    ← 中枢下沿
  GG = max(各段高点)    ← 各段高点的最高点
  DD = min(各段低点)    ← 各段低点的最低点

必要条件：ZG > ZD（否则无重叠，不构成中枢）
```

**构成方式**（从笔构建）：至少连续 3 笔的价格区间产生重叠。

- 向上笔的高点 = 终点价，低点 = 起点价
- 向下笔的高点 = 起点价，低点 = 终点价

**中枢本质**：「能量场」或「拔河地带」——后续走势要么被拉回中枢引力范围，要么积蓄足够力量突破。

**中枢定理三**：某级别的走势中枢被破坏，当且仅当一个次级别走势离开中枢后，其后的次级别回抽不重新回到该中枢内——即产生第三类买卖点。

## 7. 三类买卖点

### 7.1 第一类买卖点（背驰点）

| | 第一类买点（一买） | 第一类卖点（一卖） |
|---|---|---|
| **定义** | 下跌趋势中，最后一个中枢之后形成的底背驰点 | 上涨趋势中，最后一个中枢之后形成的顶背驰点 |
| **结构** | 离开最后一个中枢的向下次级别段力度 < 进入段力度 | 离开最后一个中枢的向上次级别段力度 < 进入段力度 |
| **性质** | 下跌趋势的绝对终结点 | 上涨趋势的绝对终结点 |
| **操作** | 全仓买入 | 全仓卖出 |

### 7.2 第二类买卖点（确认点）

| | 第二类买点（二买） | 第二类卖点（二卖） |
|---|---|---|
| **定义** | 一买出现后，首次次级别回抽不创新低的那个低点 | 一卖出现后，首次次级别反弹不创新高的那个高点 |
| **性质** | 对一买后新上升趋势的确认 | 对一卖后新下跌趋势的确认 |
| **递归** | 二买由次级别的一买构成 | 二卖由次级别的一卖构成 |

### 7.3 第三类买卖点（突破确认点）

| | 第三类买点（三买） | 第三类卖点（三卖） |
|---|---|---|
| **定义** | 次级别走势离开中枢后，回抽不回到中枢区间（低点 > ZG） | 次级别走势跌破中枢后，反弹不回到中枢区间（高点 < ZD） |
| **性质** | 中枢被向上破坏，趋势延续确认 | 中枢被向下破坏，趋势延续确认 |
| **特征** | 主升浪起爆点 | 最后逃命点 |

### 7.4 完备性定理

> **市场必然产生盈利的买卖点，只有第一、二、三类。**
> **任何向上与下跌，都必然从三类买卖点中的某一类开始以及结束。**

三类买卖点的关系：一买/一卖是转折——二买/二卖是确认——三买/三卖是延续。三者构成一个完备的分类体系，覆盖走势的一切可能状态转换。
